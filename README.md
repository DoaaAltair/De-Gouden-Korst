# De-Gouden-Korst
Bakker website De Gouden Korst
Dit project is een eenvoudige, functionele website voor de fictieve bakkerij "De Gouden Korst", gebouwd als onderdeel van een programmeeropdracht. De website is ontwikkeld met HTML, CSS, JavaScript en Cursor AI, en heeft als doel om klanten te informeren over de bakkerij, het assortiment te tonen en contact mogelijk te maken. De opdracht richtte zich op het leren gebruiken van Cursor AI, het opstellen van ontwikkelregels en iteratief ontwikkelen via Vibe-coding.

Live Demo

Bekijk de website live op: https://de-gouden-korst.vercel.app/
Projectstructuur





index.html: Home-pagina met een welkomsttekst, beschrijving van de bakkerij en een afbeelding.



assortiment.html: Overzicht van producten (brood, taart, koekjes) in een grid-layout.



contact.html: Contactformulier met velden voor naam, e-mail en bericht.



styles.css: Externe CSS voor styling van alle pagina’s, met een mobile-first aanpak.



script.js: JavaScript voor interactieve elementen, zoals een formulierbevestiging.



.cursor/rules/web-standards.mdc: Regels voor Cursor AI om consistente en toegankelijke code te garanderen.



screenshots/: Map met afbeeldingen van Cursor AI-instellingen en prompts.

Hoe de website werkt





Navigatie: Een vaste navigatiebalk bovenaan elke pagina biedt links naar Home, Ons Assortiment en Contact. De balk is gestyled met een gouden achtergrond en blijft zichtbaar tijdens het scrollen.



Home: Bevat een warme welkomsttekst, een korte beschrijving van "De Gouden Korst" en een afbeelding van een vers brood (met alt-attribuut).



Assortiment: Toont drie productcategorieën (brood, taart, koekjes) met beschrijvingen en placeholder-afbeeldingen, gerangschikt in een responsive grid.



Contact: Bevat een formulier met validatie. Bij verzending toont JavaScript een alert met "Bericht verzonden!".

De website is volledig responsive en werkt in moderne browsers (Chrome, Firefox, Safari).

Rules

Ik heb regels ingesteld in .cursor/rules/web-standards.mdc om de AI consistente en toegankelijke code te laten genereren:





Gebruik semantische HTML-tags (zoals <nav>, <main>, <footer>).



Voeg alt-attributen toe aan alle afbeeldingen.



Schrijf CSS met een mobile-first aanpak (min-width media queries).



Gebruik een warm kleurenpalet (goud: #FFD700, bruin: #8B4513) en een sans-serif lettertype (Roboto).



Deze regels zorgden ervoor dat de AI-output voldeed aan de opdrachtvereisten en W3C-standaarden.
